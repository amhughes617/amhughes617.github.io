Holy City Sherpa

A mobile first web application that acts as a tour guide for the downtown Charleston area. Users can select from 3 prebuilt tours or simply choose 3 locations around town to walk to and explore. Geofences wrap around every location and are triggered when the user comes to the location, displaying information about the location. A pitstop button displays all the restaurants nearby the user. Built using Angular.js with google apis for location data, and server code written in Java using the Spring framework. Data is persisted on a postgreSQL database, which deletes old tours after 48 hours.

  • Live site: Sherpa
  • April 2016

Iron Trivia

Iron Trivia is a web based trivia game that allows users to invite up to three other users to participate in a trivia game. The game does not start until all players are ready, and each round does not end until everyone has answered or a certain amount of time has passed. This project was made using Java and the Spring framework for the server side code, and Javascript with backbone for the client side.

  • March 2016
